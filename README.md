# Turn Based RPS (Rock, Paper, Scissors) Game Prototype
>Wow what a long title
---
This is another project of mine, prototypes to try game mechancics and such, this time it features a not so turn-based mechanic, features will be listed below such as:
- Rock, Paper, Scissors mechanics, one move beats the other.
- Switching between the said mechanics, you can change your final move before cast, since the enemy will too to confuse the player.
- Rock doesn't necessarily beat Scissors, if you do the move late or early, you still lose. Time the release.
